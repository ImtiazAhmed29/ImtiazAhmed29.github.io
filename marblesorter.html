Color-based Marble Sorter

Platform: VEX | Sensing: Reflected Light | Control: RobotC | Classification: 4/5 Reliable

Color-based marble sorting system using sensor-driven classification and servo-actuated routing.

Designed to identify and sort 5 marble categories through sensor thresholding and automated mechanical diversion. Final prototype achieved reliable classification for 4 categories.

Overview

This project was developed as an automated marble classification and routing system using the VEX platform. The machine isolates individual marbles, detects their color using reflected light intensity values, and routes them into corresponding output containers through a servo-actuated diversion mechanism.

The system was designed to classify 5 marble categories using threshold-based sensor logic, including three marbles of solid coloration and two smaller wooden and silver marble types. Final performance achieved reliable sorting for 4 categories, with silver marbles affected by overlapping sensor-response ranges and calibration sensitivity.

The separator subsystem isolated individual marbles before entering the sensing basin to ensure consistent sensor readings and prevent interference between consecutive measurements.

A 3D-printed rotating gate mechanism regulated marble flow and maintained spacing between marbles during operation. The gate would alternate between having the hole face the marble entry point and having it face downwards to allow the marble to drop.

One challenge was persistent double-feeding, which occured when a smaller marble entered the separator with a larger marble or another smaller marble. This could be avoided by making the separator hole significantly shallower, preventing marbles from stacking.

The sensing basin positioned marbles next a reflected-light sensor for threshold-based classification while temporarily holding each marble in place during sensing.

A servo-actuated pivoting track directed marbles toward the sensing region before releasing them into the routing subsystem after classification.

The sensing area was enclosed to reduce ambient light contamination and improve reading consistency between marble categories.

A limitation of this method was the unreliability of light sensing for silver marbles, likely due to the high reflectivity causing inconsistencies in the light reflected towards the sensor.

The routing basin redirected classified marbles into category-specific collection containers using a servo-driven rotational mechanism.

A central gear assembly rotated a five-cup collection system to align the appropriate output container with the marble exit path after classification.

Servo positioning angles were mapped to sensor threshold ranges corresponding to each marble category.

Technical Details

The system used threshold-based classification logic to map reflected-light sensor readings to corresponding marble categories.

Sensor values were compared against predefined ranges calibrated through repeated testing under controlled lighting conditions.

The control logic coordinated:

  • Marble release timing
  • Sensor acquisition
  • Servo-actuated routing
  • Sequential sorting operation

The sorting logic used threshold-based classification to associate sensor readings with predefined marble categories.

Marbles were tested in the sensing basin to verify color sensor values returned for different colors across multiple trials. 


task main()
{
	wait(1);
	setServo(colorServo, -60);
	setServo(sorterServo, -127);
	while(1==1){
		setServo(gateServo, 127);
		wait(1);
		setServo(gateServo, -127);
		wait(2);
		if(1000 > SensorValue[lineFollower]){ 
			setServo(sorterServo, -127);
			wait(1);
			setServo(colorServo, 100);
			wait(1);
			setServo(colorServo, -60);
		}
		else if(1800 > SensorValue[lineFollower]){ 
			setServo(sorterServo, -89);
			wait(1);
			setServo(colorServo, 100);
			wait(1);
			setServo(colorServo, -60);
		}
		else if(2100 > SensorValue[lineFollower]){ 
			setServo(sorterServo, -51);
			wait(1);
			setServo(colorServo, 100);
			wait(1);
			setServo(colorServo, -60);
		}
		else if(2700 > SensorValue[lineFollower]){ 
			setServo(sorterServo, -13);
			wait(1);
			setServo(colorServo, 100);

	}

Results and Improvements

The final prototype successfully demonstrated automated marble classification and routing using threshold-based sensing and servo-actuated mechanical diversion.

Reliable classification was achieved for 4 marble categories, while visually similar categories produced overlapping sensor response ranges under certain conditions.

Future improvements would include improved sensing methods, controlled illumination, and adaptive calibration for greater classification accuracy.
